# Number-Checker-Java
A simple Java program that takes user input and determines whether the number is positive, negative, or zero using conditional statements.
# 🔢 Number Checker in Java

## 📌 Overview

This is a beginner-friendly Java program that checks whether a given number is:

* Positive
* Negative
* Zero

It uses basic **if-else conditions** and **user input handling** with Scanner.

---

## 🚀 Features

* Takes input from the user
* Classifies numbers into positive, negative, or zero
* Simple and easy-to-understand logic
* Great for beginners learning Java

---

## 🛠️ Technologies Used

* Java
* Scanner class (java.util)

---

## ▶️ How to Run

1. Compile the program:

```
javac exam1.java
```

2. Run the program:

```
java exam1
```

3. Enter a number when prompted.

---

## 💻 Example

**Input:**

```
Enter the number you want to check:
-10
```

**Output:**

```
The number is negative
```
